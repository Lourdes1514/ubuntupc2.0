# Getting starter
```bash
	# Install NPM dependencies
	$ npm i
	
	# Simply start your app with NPM
	$ npm start
	# Build container
	$ sudo docker build --tag app-contacto .
	
	# Run container in port 8080
	$ sudo docker run --name appContacto -p 8000:3000 -d app-contacto
```
